# 📝 算法学习笔记

## 1. 螺旋矩阵（Spiral Matrix）
- **日期**：2025-01
- **核心技巧**：四个边界指针法（top/bottom/left/right）
- **思路**：每次遍历一条边 → 收缩对应的边界 → 继续下一圈
- **关键细节**：第3、4步要加 if 判断，防止单行/单列重复遍历
- **口诀**：上右下左，一圈一圈往里走

## 2. 统计奇数个数（Count Odds）
- **核心技巧**：数学公式 O(1) 解法
- **公式**：`(high + 1) // 2 - low // 2`
- **理解**：0~x 的奇数个数是 (x+1)//2，区间相减即可
- **对比**：原来自己写的 if-else 分支太多，公式法简洁很多

## 3. 原地算法（In-place Algorithm）
- **定义**：直接在原数据上修改，不复制额外空间
- **好处**：节省内存、提升效率
- **挑战**：直接修改会丢失原始信息，需要用标记法记录
